# AdNabu QA Assignment

This repository contains test cases and automation script for testing the AdNabuTestStore e-commerce website.

## Tools Used

- Python
- Selenium WebDriver
- Chrome Browser

## Project Structure

test_cases.md → Manual test cases  
test_search_add_cart.py → Automation script  
test_execution_report.md → Execution report  

## Setup Instructions

1. Install Python

2. Install Selenium
pip install selenium

3. Download ChromeDriver and add it to system PATH.

## How to Run the Test

Run the following command:

python test_search_add_cart.py

## Test Scenario Automated

Search for a product and add it to cart successfully.
